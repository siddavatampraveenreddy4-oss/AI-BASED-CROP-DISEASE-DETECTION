# AI-BASED-CROP-DISEASE-DETECTION
An AI based crop disease detection developed  using neural networks and python and opencv and flask.
which helps to detect crop disease by using leaves when the leaf will be scanned by clicking the images or live detection the infection of the leaf will be detected and the solution also provided to the disease to reduce the crop losses by using wrong pesticides and investing more money on wrong pesticides it will reduce the investment also by preventing  the diseases.
For developing this project i choose neural network and deep learining algorithms for better accuracy and high efficiency. Including with neural networks i used python libraries like numpy and pandas and opencv. I used neural netwroks algorithms those are CNN (convolutional networks),AlexNet for detecting the diseases deeply by using hidden layers.
Used opencv for captuirng image and videos and uploading those into neural networks for detection.
Used HTML and FLASK for webpage and app and website interface.
